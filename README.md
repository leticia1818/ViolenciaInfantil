# ViolenciaInfantil
O projeto é uma página web que visa conscientizar sobre a violência infantil, explicando suas formas, sinais e impactos. Ele oferece conteúdos educativos e orientações para ajudar na identificação e denúncia de casos, promovendo a proteção das crianças. A plataforma foi desenvolvida com uma interface acessível, clara e acolhedora.
